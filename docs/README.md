## 🔍 Key Concepts

### Security Layers
The transcript system uses a **7-layer defense-in-depth** architecture:

1. **Pastel Background** - Base gradient layer
2. **Guilloche Pattern** - Wave line security
3. **VOID Pantograph** - Anti-photocopy (shows "VOID" on copies)
4. **Copy Detection** - Micro-dot pattern
5. **Latent Images** - Angle-dependent patterns
6. **Microprint** - Repeating security text
7. **Holographic Overlay** - Color-shifting gradient

### Microtext Layers
Three levels of micro-text for authentication:

- **Primary:** 2.5-3px (visible with magnification)
- **Hidden:** 1.5px "AUTHENTIC" watermark
- **Forensic:** 1px tracking ID (invisible to naked eye)

### Verification Methods
Three ways to verify transcript authenticity:

1. **Quick Check** (30 seconds) - 5 critical features
2. **Full Verification** (3 minutes) - 15-point checklist
3. **Forensic Analysis** (15 minutes) - Lab techniques
